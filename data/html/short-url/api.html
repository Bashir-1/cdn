短链生成

您使用本站的API就视为您同意以下用户协议，若您不同意，请不要使用。

不管是以下哪个域名，数据库都是通用的
请按照以下格式请求，非法请求将会遭到防火墙阻断
由于防火墙启用速率限制，一般正常情况使用将不会遭到误判，若请求太快可能会造成误判
本短链服务不要求http或https，所有http请求将会被转接到https
若要使用本短链服务，请注明来源，谢谢

短链API支持服务

s.spacemy.cn

# POST /create
curl -X POST -H "Content-Type: application/json" -d '{"url":"https://s.spacemy.cn"}' https://s.spacemy.cn/create

# 指定slug
curl -X POST -H "Content-Type: application/json" -d '{"url":"https://s.spacemy.cn","slug":"scxs"}' https://s.spacemy.cn/create

response:

{
  "slug": "<slug>",
  "link": "https://s.spacemy.cn/<slug>"
}
